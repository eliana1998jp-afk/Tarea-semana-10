Backlog priorizado

1. Como ciudadano quiero registrar una incidencia para dejar evidencia del caso.

2. Como coordinador quiero recibir una notificación en Telegram cuando se registre una nueva incidencia para darle seguimiento oportuno.

3. Como miembro del equipo quiero consultar los reportes registrados para verificar el estado de cada caso.

4. Como administrador quiero actualizar el estado de los reportes para mantener la información organizada.

5. Como ciudadano quiero conocer el estado de mi reporte para saber si ya fue atendido.
